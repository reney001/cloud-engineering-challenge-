 Cloud Engineering Challenge

 Answers

1. What is cloud computing?
Cloud computing is the delivery of computing services (servers, storage, databases, networking, software, etc.) over the internet ("the cloud") instead of using local servers or personal computers. It allows users to access resources on-demand without managing physical hardware.

 2. What is DevOps?
DevOps is a set of practices that combines software development (Dev) and IT operations (Ops) to shorten the development lifecycle and deliver high-quality software continuously. It emphasizes automation, collaboration, and continuous integration/continuous deployment (CI/CD).

3. Difference between Docker and Kubernetes?
- Docker is a platform used to build, run, and package applications into lightweight containers.
- Kubernetes (K8s) is an orchestration tool that manages, scales, and automates the deployment of multiple containers across multiple servers.

Simple explanation: Docker puts your app in a box. Kubernetes manages many boxes at scale.

 4. Why are Linux skills important in cloud engineering?
Most cloud servers and services (AWS, GCP, Azure) run on Linux. Strong Linux skills are essential for:
- Server administration
- Writing shell scripts
- Managing containers
- Troubleshooting infrastructure
- Working with cloud CLI tools
